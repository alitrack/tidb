# tidb
TiUP playground


## TiSpark Test Enviroment

To test TiSpark, please run 

```
chmod +x init.tispark

./init.tispark
```

the spark.sql is sql for testing tispark easily, 

```
./spark/bin/spark-sql -f spark.sql
```
